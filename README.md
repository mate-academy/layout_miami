# Miami landing page
## Adaptive design for:
- Large screens 2560px
- Full HD 1920px
- The design 1600px
- Notebook 1280px
- Tablet 1024
- Mobile (> 320px)

## Tech stack
- CSS (SASS)
- HTML
- BEM

## Install / run
Use 'npm install' to install project and 'npm run' to run project
