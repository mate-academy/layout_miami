
# MyBike Landing Page

Simple MyBike Landing page with Contact Us form.

## Demo

https://sapnachoudhary06.github.io/layout_miami/

## Run Locally

Clone the project

```bash
  git clone https://github.com/sapnachoudhary06/layout_miami.git
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
