# Miami landing page
Implemented landing page according to [Figma design](https://www.figma.com/file/nHz8bflIwJaWP3P99vKTH5/miami_home_new?node-id=0%3A2)

Full responsive for:
- Large screens 2560px
- Full HD 1920px
- The design 1600px
- Notebook 1280px
- Tablet 1024
- Mobile (> 320px)

Technology used: HTML, Flexbox, Grid, SCSS, BEM methodology
